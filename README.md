#--------------battleship-------------
1. -This project is a two-player Battleship game built using HTML, CSS, and JavaScript.
2. -The game is played locally by two players who take turns placing ships and shooting on the opponent’s board.  
3. -The project focuses on game logic, state management, and DOM manipulation.

#--------------How-to-Play------------
1. -Player 1 places their ships on the grid.
2. -Player 2 places their ships on the grid.
3. -Each player has a total of 9 ship cells (ships of size 1, 3, and 5).
4. -After setup, players take turns shooting at the opponent’s board.
5. -A hit is marked when a ship cell is clicked.
6. -When a player has no remaining ship cells, the game ends and a winner is declared.
7. -The Reset button restarts the game.
